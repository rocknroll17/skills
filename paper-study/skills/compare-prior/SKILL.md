# 선행작 비교

지정한 선행 연구를 본 논문과 비교. CLAUDE.md §5 준수 (표 사용, 핵심 차이 열만).

## 1. 비교표 (markdown table)

비교 축은 **논문 domain에 맞게** 선택. 예시 (각자 논문 맥락에 맞춰 교체):

| 축 | 본 논문 | 선행작 A | 선행작 B |
| --- | --- | --- | --- |
| 대상 범주 | ... | ... | ... |
| 백본 / 핵심 모듈 | ... | ... | ... |
| 학습 데이터 | ... | ... | ... |
| 주요 loss / 학습 전략 | ... | ... | ... |
| 한계 | ... | ... | ... |

해당 없는 축은 "N/A".

## 2. 핵심 차별점 (1~2문장)
- 이 비교에서 본 논문이 **근본적으로 새로운 것**은?
- 반대로 선행작이 **우위인 영역**이 있다면 솔직히 표기.

## 3. 계보 한 줄
```
Prior_1 → Prior_2 → ... → 본 논문
```
각 화살표가 "무엇을 이어받고 무엇을 바꿨는지" 한 문장씩.

## 주의

- 논문에 **명시된** 비교만 사용. 추측·숫자 조작 금지.
- 모든 수치는 출처(논문 표·페이지) 병기.
- 인터넷 조사가 필요하면 WebSearch → 출처 URL 병기.
- 전체 300~400단어 이내. 브리핑 모드.
